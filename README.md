# NetWorthTracker
Simple Net Worth Tracker in Kotlin.



## Run Locally

Build project (then you can use the net-worth-tracker script)

```bash
  ./gradlew installDist
```



Terminal Preview

```bash
  ./net-worth-tracker
```

Start the server for Web Preview (then go to localhost:8080)

```bash
  ./net-worth-tracker --web
```
